# Ex. 2 - Search Field

## Instructions

Dans un premier temps, nous allons nous concentrer sur le champ de recherche, un simple `input` contrôlé, nous afficherons la recherche tapée par l'utilisateur.

Nous utiliserons [Smooth UI](https://smooth-ui.smooth-code.com/docs-components-input), une librairie d'UI moderne et simple. Cela nous permettra d'avoir un site agréable à regarder sans avoir à écrire beaucoup de CSS.

Smooth UI utilise [styled-components](https://www.styled-components.com/), la solution la plus populaire pour faire du CSS in JS en React.

- Installer `styled-components @smooth-ui/core-sc`
- Créer un composant `input` contrôlé dans `src/components/SearchInput.js`
- Placer le composant `SearchInput` dans `App` et afficher sa valeur à l'aide de `useState`
- Supprimer les fichiers inutilisés : `App.css`, `logo.svg` et `App.test.js`

### Résultat attendu

Un champs de recherche simple, avec la recherche qui reflète ce qu'a tapé l'utilisateur.

### Installation d'un package npm

```bash
# Installation de packages depuis npm
npm i styled-components @smooth-ui/core-sc
```

### SearchInput.js

```js
// SearchInput.js
import React from 'react'
import { Input } from '@smooth-ui/core-sc'

export default function SearchInput(props) {
  return (
    <Input
      placeholder="Type something..."
      width="100%"
      type="search"
      {...props}
    />
  )
}
```

### Récupération de la valeur d'un input

```js
<input
  onChange={event => {
    // Un event est passé en paramètre, "target" désigne l'élément DOM "input"
    // Un "input" a une propriété "value" qui correspond à la valeur qu'il contient
    console.log(event.target.value)
  }}
/>
```
